# HEARTBEAT.md - Periodic Checks

Quick health check tasks to run on heartbeat polls.

## When to Run

Rotate through these checks (2-4 times per day):
- Morning (08:00-10:00)
- Midday (12:00-14:00)
- Afternoon (16:00-18:00)
- Evening (20:00-22:00)

## Checklist

### 1. Communications
- [ ] Any unread emails?
- [ ] Calendar events in next 24h?
- [ ] WhatsApp messages needing attention?

### 2. Memory Maintenance
- [ ] Review recent `memory/YYYY-MM-DD.md` files
- [ ] Update `MEMORY.md` with significant events
- [ ] Clear outdated entries

### 3. Workspace Health
- [ ] Git status - uncommitted changes?
- [ ] Any files needing organization?

### 4. External World
- [ ] Weather relevant for next 12h?
- [ ] Any news/sources to check?

## When to Reach Out

- Important email/calendar event
- Event starting <2h
- Something interesting discovered
- >8h since last message
- User explicitly asked for updates

## When to Stay Quiet (HEARTBEAT_OK)

- Late night (23:00-08:00) unless urgent
- Nothing new since last check
- Just checked <30 min ago
- User clearly busy

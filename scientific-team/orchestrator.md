2. Economics & Resource Agent
3. Governance & Ethics Agent

5. Health & Pandemic Agent

- Copernicus Climate Service

**Integration:** All agents report to SafeProactive for proposal management. SPEACE core (DigitalDNA+SMFOI) handles self-evolution.
